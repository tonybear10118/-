###              程式說明                ###
############################################

此程式被設計用來爬起Dcard上面文章裡的圖片

介面說明:
	-> 說明按鈕 : 開啟說明程式文字檔
	-> 西斯板 :   開始爬取Dcard西斯板文章
	-> 攝影板 :   開始爬取Dcard攝影板文章
	-> 寵物板 :   開始爬取Dcard寵物板文章

  在爬文章前可做個人化設定 :
	-> 切換性別 : 只選取特定性別所發的文章(也可全部都爬)
	-> 增加文章數量 : 如果覺得爬取的數量不夠多可以增加爬取的文章數量
	-> 減少文章數量 : 如果覺得爬取的數量太多可以減少爬取的文章數量

  狀態顯示:
	-> 共下載了 x 張 (單次已下載圖片張數)
	-> 狀態歷程 : log供使用者查看
	-> 停止爬蟲 : 若途中想停止爬蟲動作即可按此按鈕停止

  路徑及退出:
	-> 瀏覽下載路徑 : 可供使用者選擇下載的檔案位置
	-> xx:\xx\xx : 目前會把檔案下載下來的位置
	-> 退出 : 按下即可退出程式
